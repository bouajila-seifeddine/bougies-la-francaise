PREPARATION
--------
Login to FTP server and go to your PrestaShop web directory and unpack delivered facebooktrackingpixel package into modules directory.
After unpacking of the package the following directory structure must be create under "modules" subdirectory:

- modules
  |- facebooktrackingpixel
  |   |- css
  |   |   |- index.php
  |   |   |- facebooktrackingpixel.css
  |   |- img
  |   |   |- index.php
  |   |   |- facebooktrackingpixel.png
  |   |- views
  |   |   |- templates
  |   |       |- front
  |   |           |- order_confirmation.tpl
  |   |- translations
  |       |- index.php
  |   |   |- fr.php
  |   |- index.php
  |   |- config.xml
  |   |- facebooktrackingpixel.php
  |   |- logo.gif
  |   |- logo.png
  |   |- readme_en.txt
  |- ...
  
INSTALLATION
--------
Login to the PrestaShop administration panel and install the Facebook Tracking Pixel module. 
Configure module with the information related to your Facebook settings - tracking pixel id, etc.

YOU ARE DONE
--------
