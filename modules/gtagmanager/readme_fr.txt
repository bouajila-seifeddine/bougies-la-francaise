Google Tag Manager - Un module Prestashop par Sutunam (http://sutunam.com)
---------------------------
# Notes #

- Ce module recquiert un compte Google pour fonctionner (http://www.google.com/)
- En cas de problème, contactez-nous à cette adresse : support@sutunam.com

# Installation #

	> Avec l'interface Modules

1. Allez dans votre panneau d'administration Prestashop, Modules > Ajouter un module
2. Sélectionnez le ficher .zip contenant le module
3. Cliquez sur le bouton Charger le module
4. Une fois l'envoi terminé, rendez-vous dans la catégorie Statistiques & Analyses de la page Modules, trouvez le module Google Tag Manager, et cliquez sur installer. 

	> Avec le protocole FTP

1. Connectez-vous via FTP sur votre serveur
2. Allez à la racine de votre boutique Prestashop
3. Collez le dossier gtagmanager décompressé dans le dossier /modules/ 
4. Une fois l'envoi terminé, rendez-vous dans la catégorie Statistiques & Analyses de la page Modules, trouvez le module Google Tag Manager, et cliquez sur installer. 

# Configuration #

1. Rendez vous dans Modules > Statistiques et Analyses > Google Tag Manager
2. Cliquez sur le bouton Configurer
3. Collez votre code GTM dans la boîte de texte, enregistrez.
4. Votre module est prêt à fonctionner
