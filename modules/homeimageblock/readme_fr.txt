
********************************
Description
********************************
Ce module vous permet d'afficher des blocks d'image sur votre home sous forme de layout responsive et fluid grid avec des liens vers différents type 
de contenu : CMS, Produit, Catégory, Résultat de recherche.


********************************
Les +
********************************
Le client pourra visualiser de manière directe sur votre home des push vers différents types de contenus


********************************
Installation
********************************
Ce module est uniquement valable pour la version 1.5.3 et supérieur de Prestashop.

Il gère le multilangue et le multiboutique.

Installer votre module normallement. L'administration en back Office se trouve sous Catalogue/Image Accueil.


********************************
Fonctionnement
********************************
Vous créez des blocks d'image dans l'administration. Ces images pourront avoir un lien associé qui pourra pointer vers une page de CMS,
Un produit, une catégorie, un formulaire... Ces blocks d'images peuvent être différents selon la langue et la boutique. 
Vous avez également la possibilité d'ordonner vos images dans un ordre d'affichage, de les espacer les une par rapport aux autres,
de spécifier une animation au survol...
En front, vos images vont se placer les une à coté des autres dans l'ordre définit. Ce placement se fera sous forme de grille fluid tel
qu'on peut l'obtenir avec des plugions jquery tel que Isotop ou encore Jquery Nested.


********************************
Fonctionnalités
********************************
Gére le multilangue
Gère le mutiboutique
Gère les positions des images
Gestion des marges entre les images
Possibilité d'animer les images
Possibilité de désactiver certaines images
